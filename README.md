## 👩‍💻 実績
### 🔸 人材紹介会社｜WordPress固定ページ追加
💡 ざっくりとした指示書から、気づきや不明点は丁寧にコミニケーションをとり対応しました。
- 既存の業種別の紹介ページなどを参考に、都道府県別の複数ページを新規で作成。
- 合わせて、既存ページのレイアウト修正なども対応。
- メニューに都道府県別のページを追加するために、メニューのレイアウトの調整も対応。

### 🔸 導入事例紹介｜カスタムフィールドの実装
💡 記事の投稿者にとって分かりやすいカスタムフィールドのUIを心がけました。
- プラグイン「Advanced Custom Fields」にてカスタム投稿およびカスタムフィールドの機能を新規で実装。
- ループ処理でアーカイブページを実装。
- インタビュー形式の導入事例で、記事により質問と回答の回数など形式が異なっていても汎用的に利用できるよう、UIを工夫。

### 🔸 採用サイト｜WordPress固定ページ追加
- 会社説明会のページを新規で追加。
- 説明会の申し込み用に、Googleフォームを埋め込み。
